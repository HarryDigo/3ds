# Tarefa google maps

Aplicativo simples, com um mapa e a capacidade de pesquisar endereços/locais

## Execução

1. Clonar o repositório
```bash
git clone https://github.com/HarryDigo/3ds
```

2. Navegar até a pasta desta tarefa
```bash
cd 3ds/pam/native_4/maps-pam
```

3. Criar o arquivo das váriaveis de ambiente
```bash
echo "EXPO_PUBLIC_GOOGLE_MAPS_API_KEY=<sua-chave-api>" > .env
```

4. Instalar as dependências
```bash
npm i
```

5. Executar o projeto (adicionar --tunnel caso os dispositivos estejam em redes diferentes)
```bash
npx expo start
```
